# EDA Projects

A collection of data analysis projects developed as part of my ICT Business Analysis 
studies. Each project applies structured analytical techniques to real-world business 
datasets — covering sales performance, marketing effectiveness, and public health data 
— with the goal of translating raw data into actionable business insights.

All projects are built using Python for analysis and Tableau for visualization.

---

## Projects

| Project | Description |
|---|---|
| Balaji Fast Food Sales | Sales and revenue analysis of a fast food restaurant |
| Multiple Restaurant Sales | Cross-branch performance and manager benchmarking |
| Marketing Campaign | ROI and channel effectiveness across campaign types |
| COVID-19 Data | Global case and mortality trend analysis |
